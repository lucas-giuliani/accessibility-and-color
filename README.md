# Color, percepción y accesibilidad

Este repositorio reúne exploraciones sobre color, percepción visual, daltonismo y accesibilidad aplicadas al diseño de productos y experiencias.

El color nunca aparece aislado. Convive con contexto, contraste, jerarquía, hábitos culturales y decisiones de interfaz. Por eso, diseñar con color no debería reducirse a elegir una paleta atractiva o respetar un manual de marca.

La intención de este espacio es revisar cómo usamos el color, qué información depositamos en él y qué pasa cuando distintas personas no perciben una interfaz de la misma manera.

## Temas

- accesibilidad visual;
- contraste y legibilidad;
- daltonismo y deficiencias en la percepción del color;
- color como señal de estado;
- jerarquía visual;
- diseño inclusivo;
- percepción y comportamiento;
- herramientas de simulación y validación;
- decisiones de producto que dependen demasiado del color.

## Una pregunta de partida

Muchas interfaces usan el color como si todas las personas lo interpretaran de la misma forma.

El verde confirma.  
El rojo advierte.  
El amarillo llama la atención.  
El gris parece deshabilitado.

Pero cuando la información depende únicamente del color, una decisión aparentemente estética puede convertirse en una barrera.

Este repositorio busca documentar ejemplos, pruebas y criterios para diseñar señales que no dependan de una única forma de percepción.

## Qué voy a incluir

- artículos y reflexiones;
- ejemplos comparativos;
- simulaciones de distintos tipos de daltonismo;
- errores frecuentes;
- decisiones de diseño;
- recursos y herramientas;
- pequeñas guías prácticas;
- material visual complementario.

## Contenidos

Este repositorio está en construcción.

La primera exploración va a estar dedicada a cómo el color puede modificar la percepción de una misma imagen y por qué eso debería importarnos al diseñar interfaces, servicios y comunicaciones.
